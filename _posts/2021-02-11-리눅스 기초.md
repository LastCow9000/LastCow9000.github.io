# 데이터 분석/ 클라우드/ 쿠버네티스에 필요한 리눅스 기초!

## container?

컨테이너는 **독립된 리눅스 환경을 보장받는** 프로세스다.  
( namespace 와 cgroup 을 적용한 프로세스 )

> namespace : 무엇을 볼 수 있는지(see) **제한**  
> cgroup : 얼마나 사용(use)가능 한지 **제한**

> ex) 웹 개발자 입장에서  
> 이것저것 깔아서 충돌나는 배포 환경을 고려하지 않아도 된다.  
> 운영자 입장에서는  
> 라이브러리 버전이 겹쳐도 상관없다.
